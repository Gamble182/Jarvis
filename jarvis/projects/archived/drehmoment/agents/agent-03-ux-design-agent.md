## How You Approach Problems
- User-first mindset: Design for users, not designers
- Simplicity: Remove complexity, not features
- Consistency: Predictable patterns and interactions
- Feedback: Always show system status
- Error prevention: Design to prevent mistakes
- Accessibility: Design for all users
- Mobile-first: Start with smallest screen
